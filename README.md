# 🎓 Student Result Management System (DBMS Project)

A **Flask-based web application** that allows administrators, faculty, and students to manage and view student academic records.  
The system supports **user authentication, role-based dashboards, student data management, and Excel-based mark uploads**.

This project demonstrates the use of **Python, Flask, SQLAlchemy, SQLite, and HTML/CSS** for building a database-driven web application.

---

## 🚀 Features

### 🔐 Authentication System
- Secure login system
- Password hashing using `Werkzeug`
- Role-based access:
  - **Admin**
  - **Faculty**
  - **Student**

---

### 👨‍💼 Admin Dashboard
Admin can:

- Add and manage **students**
- Add and manage **faculty**
- Edit or update user information
- View overall system data
- Manage login accounts

---

### 👨‍🏫 Faculty Dashboard
Faculty can:

- View assigned students
- Upload student marks using **Excel files**
- Update student marks
- View student rankings
- Manage student academic records

---

### 👨‍🎓 Student Dashboard
Students can:

- View their **profile**
- Check **marks**
- View **rankings**
- Access academic performance data
